Ejercicio #2: PROMEDIO DE 3 NÚMEROS

Escriba un algoritmo que pregunte por 3 números y muestre como resultado el promedio de estos. Use Pseudocódigo y diagrama de flujo.

Pseudocódigo

Variables
Nombre Descripción Tipo
n1 Primer valor a sumar Int
n2 Segundo valor a sumar Int
n3 Tercer valor a sumar Int
respuesta Resultado de la suma y división Int


Proceso

  1. Inicio
  2. Leer "n1", "n2" y "n3"
  3. Hacer "respuesta" = "n1" + "n2" + "n3" / 3
  4. Escribir "respuesta"
  5. Fin
