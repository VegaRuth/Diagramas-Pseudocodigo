Ejercicio #1: SUMA DE DOS NÚMEROS

Escriba un algoritmo que pregunte por dos números y muestre como resultado la suma de estos. Use Pseudocódigo y diagrama de flujo.

Pseudocódigo

Variables
Nombre Descripción Tipo
n1 Primer valor a sumar Int
n2 Segundo valor a sumar Int
respuesta Resultado de la suma Int


Proceso

  1. Inicio
  2. Leer "n1" y "n2"
  3. Hacer "respuesta" = "n1" + "n2"
  4. Escribir "respuesta"
  5. Fin
